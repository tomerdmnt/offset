# offset

  Get element offset

  ```javascript
var offset = require('offset');

var o = offset(document.querySelector('div'));
console.log('x: ' + o.x);
console.log('y: ' + o.y);
  ```

## Installation

    $ component install tomerdmnt/offset

## API

### offset(el)
gets an element. returns an object with properties x and y.
